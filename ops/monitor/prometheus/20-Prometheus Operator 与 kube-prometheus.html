                                # Prometheus Operator

                                еңЁ Kubernetes дёҠз®ЎзҗҶ Prometheus йӣҶзҫӨпјҢдҪҝз”Ё Kubernetes иҮӘе®ҡд№үиө„жәҗпјҲCRDпјүпјҢз®ҖеҢ–дәҶ PrometheusгҖҒAlertmanager е’Ңзӣёе…ізӣ‘жҺ§з»„д»¶зҡ„йғЁзҪІе’Ңй…ҚзҪ®гҖӮзӣ®еүҚжҸҗдҫӣеӣӣз§Қиө„жәҗзұ»еһӢ

                                • PrometheusпјҡеЈ°жҳҺејҸеҲӣе»әе’Ңз®ЎзҗҶPrometheus Serverе®һдҫӢ
                                • ServiceMonitorпјҡиҙҹиҙЈеЈ°жҳҺејҸзҡ„з®ЎзҗҶзӣ‘жҺ§й…ҚзҪ®
                                • PrometheusRuleпјҡиҙҹиҙЈеЈ°жҳҺејҸзҡ„з®ЎзҗҶе‘ҠиӯҰй…ҚзҪ®
                                • AlertmanagerпјҡеЈ°жҳҺејҸзҡ„еҲӣе»әе’Ңз®ЎзҗҶAlertmanagerе®һдҫӢ

                                з®ҖиЁҖд№ӢпјҢPrometheus OperatorиғҪеӨҹеё®еҠ©з”ЁжҲ·иҮӘеҠЁеҢ–зҡ„еҲӣе»әд»ҘеҸҠз®ЎзҗҶPrometheus Serverд»ҘеҸҠе…¶зӣёеә”зҡ„й…ҚзҪ®гҖӮ

                                # CRD

                                Prometheus Operator зҡ„дёҖдёӘж ёеҝғзү№жҖ§жҳҜ watch Kubernetes API жңҚеҠЎеҷЁеҜ№зү№е®ҡеҜ№иұЎзҡ„жӣҙж”№пјҢе№¶зЎ®дҝқеҪ“еүҚ Prometheus йғЁзҪІдёҺиҝҷдәӣеҜ№иұЎеҢ№й…ҚгҖӮOperator еҜ№д»ҘдёӢиҮӘе®ҡд№үиө„жәҗе®ҡд№ү (crd) иҝӣиЎҢж“ҚдҪңпјҡ

                                monitoring.coreos.com/v1:

                                • Prometheus: е®ғе®ҡд№үдәҶ Prometheus жңҹжңӣзҡ„йғЁзҪІгҖӮ
                                • Alertmanager: е®ғе®ҡд№үдәҶ AlertManager жңҹжңӣзҡ„йғЁзҪІгҖӮ
                                • ThanosRuler: е®ғе®ҡд№үдәҶ ThanosRuler жңҹжңӣзҡ„йғЁзҪІпјӣеҰӮжһңжңүеӨҡдёӘ Prometheus е®һдҫӢпјҢеҲҷйҖҡиҝҮ ThanosRuler иҝӣиЎҢе‘ҠиӯҰи§„еҲҷзҡ„з»ҹдёҖз®ЎзҗҶгҖӮ
                                • ServiceMonitor: Prometheus Operator йҖҡиҝҮ PodMonitor е’Ң ServiceMonitor е®һзҺ°еҜ№иө„жәҗзҡ„зӣ‘жҺ§пјҢServiceMonitor з”ЁдәҺйҖҡиҝҮ Service еҜ№ K8S дёӯзҡ„д»»дҪ•иө„жәҗиҝӣиЎҢзӣ‘жҺ§пјҢжҺЁиҚҗйҰ–йҖү ServiceMonitor. е®ғеЈ°жҳҺжҖ§ең°жҢҮе®ҡдәҶ Kubernetes service еә”иҜҘеҰӮдҪ•иў«зӣ‘жҺ§гҖӮOperator ж №жҚ® API жңҚеҠЎеҷЁдёӯеҜ№иұЎзҡ„еҪ“еүҚзҠ¶жҖҒиҮӘеҠЁз”ҹжҲҗ Prometheus еҲ®ж“Ұй…ҚзҪ®гҖӮ
                                • PodMonitor: Prometheus Operator йҖҡиҝҮ PodMonitor е’Ң ServiceMonitor е®һзҺ°еҜ№иө„жәҗзҡ„зӣ‘жҺ§пјҢPodMonitor з”ЁдәҺеҜ№ Pod иҝӣиЎҢзӣ‘жҺ§пјҢжҺЁиҚҗйҰ–йҖү ServiceMonitor. PodMonitor еЈ°жҳҺжҖ§ең°жҢҮе®ҡдәҶеә”иҜҘеҰӮдҪ•зӣ‘и§ҶдёҖз»„ podгҖӮOperator ж №жҚ® API жңҚеҠЎеҷЁдёӯеҜ№иұЎзҡ„еҪ“еүҚзҠ¶жҖҒиҮӘеҠЁз”ҹжҲҗ Prometheus еҲ®ж“Ұй…ҚзҪ®гҖӮ
                                • Probe: е®ғеЈ°жҳҺжҖ§ең°жҢҮе®ҡдәҶеә”иҜҘеҰӮдҪ•зӣ‘и§Ҷ ingress жҲ–йқҷжҖҒзӣ®ж Үз»„гҖӮOperator ж №жҚ®е®ҡд№үиҮӘеҠЁз”ҹжҲҗ Prometheus еҲ®ж“Ұй…ҚзҪ®гҖӮ
                                • PrometheusRule: з”ЁдәҺз®ЎзҗҶ Prometheus е‘ҠиӯҰи§„еҲҷпјӣе®ғе®ҡд№үдәҶдёҖеҘ—жүҖйңҖзҡ„ Prometheus иӯҰжҠҘе’Ң/жҲ–и®°еҪ•и§„еҲҷгҖӮPrometheus з”ҹжҲҗдёҖдёӘи§„еҲҷж–Үд»¶пјҢеҸҜд»Ҙиў« Prometheus е®һдҫӢдҪҝз”ЁгҖӮ
                                • AlertmanagerConfig: з”ЁдәҺз®ЎзҗҶ AlertManager й…ҚзҪ®ж–Үд»¶пјҢдё»иҰҒжҳҜе‘ҠиӯҰеҸ‘з»ҷи°Ғпјӣе®ғеЈ°жҳҺжҖ§ең°жҢҮе®ҡ Alertmanager й…ҚзҪ®зҡ„еӯҗйғЁеҲҶпјҢе…Ғи®ёе°ҶиӯҰжҠҘи·Ҝз”ұеҲ°иҮӘе®ҡд№үжҺҘж”¶еҷЁпјҢе№¶и®ҫзҪ®зҰҒжӯўи§„еҲҷгҖӮ

                                Prometheus Operator иҮӘеҠЁжЈҖжөӢ Kubernetes API жңҚеҠЎеҷЁеҜ№дёҠиҝ°д»»дҪ•еҜ№иұЎзҡ„жӣҙж”№пјҢе№¶зЎ®дҝқеҢ№й…Қзҡ„йғЁзҪІе’Ңй…ҚзҪ®дҝқжҢҒеҗҢжӯҘгҖӮ

                                3. ејҖз®ұеҚіз”Ёзҡ„еӨ§йҮҸпјҡ
                                  1. зӣ‘жҺ§еҜ№иұЎпјҢеҰӮпјҡK8S з»„д»¶ - coredns, kubelet, controller manager, apiserver, etcd, scheduler, kube proxy; зӣ‘жҺ§з»„д»¶иҮӘзӣ‘жҺ§ - grafana, AlertManager, prometheus зӯүпјӣ
                                  2. д»ӘиЎЁжқҝпјҢиҮӘеёҰ 24 дёӘд»ӘиЎЁжқҝпјҢйқһеёёе®һз”ЁпјҢж¶өзӣ–пјҡйӣҶзҫӨ/з»„д»¶/зҪ‘з»ң/еӯҳеӮЁ/Node/Pod зӯүзӯүз»ҙеәҰпјӣ
                                  3. е‘ҠиӯҰи§„еҲҷпјҢиҮӘеёҰдәҶ 100 еӨҡдёӘе‘ҠиӯҰи§„еҲҷпјҢж¶өзӣ– K8S зҡ„ж–№ж–№йқўйқўпјӣ
                                4. жөҒиЎҢзҡ„ејҖжәҗдә§е“ҒпјҢеҫҲеӨҡд№ҹй»ҳи®ӨдјҡеёҰжңүеҜ№ Prometheus Operator зҡ„ж”ҜжҢҒпјҢеҰӮ Loki е°ұжңүзӣёе…ізҡ„ ServiceMonitor;
